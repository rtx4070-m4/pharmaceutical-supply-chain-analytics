# Pharmaceutical Supply Chain Analytics

## Project Overview
This project analyzes pharmaceutical supply chain data to identify delivery bottlenecks,
supplier performance issues, and inventory trends. The goal is to improve healthcare
logistics and optimize pharmaceutical distribution.

## Tools Used
- Python
- SQL
- Tableau

## Project Workflow
Data Collection → Data Cleaning → SQL Analysis → Python EDA → Tableau Dashboard

## Key Analysis
- Supplier delivery performance
- Pharmaceutical product demand
- Inventory levels across warehouses
- Transportation mode efficiency
- Supply chain cost analysis
